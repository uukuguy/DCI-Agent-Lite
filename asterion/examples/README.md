# Asterion application examples

`applications/dci_research.py` demonstrates explicit assembly, catalog, runtime,
and implementation binding. `applications/controlled_code.py` demonstrates
explicit host-service injection.

These files are repository examples. They are neither installed nor discovered
as application providers. Normal users run the `asterion` or `asterion-dci`
commands instead.

End-to-end executable DCI examples live under `scripts/examples/` and are
available through `make example` and `make runtime-example`.
